love.update (dt)
Update everything. If the time elapsed between two consecutive updates is greater than 0.3s, the update is skipped. This is useful to avoid glitches when dragging the screen around or temporary drops in frame rate.

Parameters:

  • dt: Time elapsed since the last update, in seconds.
math.clamp (input, min_val, max_val)
Clamp number between two values. Returns input if it is between min_val and max_val; max_val if it is larger than it; min_val if it is smaller than it.

Parameters:

  • input: The value to be clamped
  • min_val: The lower bound of the clamping interval.
  • max_val: The upper bound of the clamping interval. Must be greater than or equal to min_val
