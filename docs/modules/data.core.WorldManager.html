WorldManager:pauseGame (time)
Pauses the game. While the game is paused, nothing moves by itself: no update functions are called. the game will be paused until WorldManager:unPauseGame is called.

Parameters:

  • time (Optional) The time for which the game will be paused, in seconds. If omitted,
WorldManager:unPauseGame ()
Unpauses the game. Update functions will start being called again after this function is called.
onDynamicCollide (dt, shapeA, shapeB)
Called when two active elements collide.
If the collision needs to be resolved in a particular order, this should be the place to decide it. For now there is no guarantee on which of the two objects is the first to resolve the collision.

Parameters:

  • dt The time slice for the frame when the collision is detected
  • shapeA The first colliding shape, as a HardonCollider shape.
  • shapeB The second colliding shape, as a HardonCollider shape.
