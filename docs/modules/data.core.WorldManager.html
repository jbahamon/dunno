WorldManager:fadeToColor (rampTime, color, numSteps)
Fades the screen to a solid tint. having discrete intermediate values. If omitted, the transition is done as smoothly as possible.

  • rampTime The time (in seconds) for the screen to get to the tint's full intensity.
  • color The color to apply to all elements, as an array (not a dictionary) of four values, corresponding to red, green, blue, and alpha, in that order.
