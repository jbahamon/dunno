loadStage (name)

    Stage The newly created Stage
loadCharacter (name)
Loads a Player character, given its folder name.
Transform, Input and Physics components are added to the GameObject. Animation, Collision and StateMachine components are added if sprites (and animations), size and states (or basicStates) fields are present in the character’s parameters.

Parameters:

  • name string The Player character’s folder’s name.

Returns:

    GameObject The newly created Player character.
loadBasicStates (object, stateParams)
Adds a set of basic player states to a GameObject. The GameObject should have a StateMachine, Animation, Transform, Input, Collision and Physics components. See the parameter specification (TODO!) for details of building a GameObject from a set of parameters.

Parameters:

  • object GameObject The GameObject that will receive the created states. Must have a StateMachineComponent.
  • stateParams table The parameter table.
loadStates (object, states)
Creates and adds states from a list of state parameters. See the parameter specification (TODO!) for details of building a GameObject from a set of parameters.

Parameters:

  • object GameObject The GameObject that will receive the created states. Must have a StateMachineComponent.
  • states table The states' parameters table.
normalizeDynamics (dynamics)
Sets the default parameters for a dynamics table if they're missing.

Parameters:

  • dynamics table The dynamics table to normalize.
loadTransitions (object, transitions)
Creates and adds transition from a parameter table. See the parameter specification (TODO!) for details of building an Element from a set of parameters.

Parameters:

  • object GameObject The GameObject that will receive the created transitions. Must have a StateMachineComponent.
  • transitions table The transitions' parameters table.
