Functions

getCollisionArea (box1, box2)
Calculates the contact area between two rectangles (specifically, two HardonCollider rectangle shapes.

Parameters:

  • box1 Shape The first HardonCollider rectangle.
  • box2 Shape The second HardonCollider rectangle.

Returns:

    number The contact area, in units^2 (where the shapes' positions are measured in units)
isBoxInRange (box, topLeft, bottomRight)
Calculates whether a rectangle (specifically, a HardonCollider rectangle shape) intersects a certain range or not.The frontier of the range is considered as part of it.

Parameters:

  • box Shape The rectangle HardonCollider to check.
  • topLeft vector The position of the upper-left corner of the range to check.
  • bottomRight vector The position of the bottom-right corner of the range to check.

Returns:

    bool Whether the box is in the specified range
isPointInRange (point, topLeft, bottomRight)
Calculates whether a point is in a certain range or not. The frontier of the range is considered as part of it.

Parameters:

  • point vector The point to check.
  • topLeft vector The position of the upper-left corner of the range to check.
  • bottomRight vector The position of the bottom-right corner of the range to check.

Returns:

    bool Whether the point is in the specified range
