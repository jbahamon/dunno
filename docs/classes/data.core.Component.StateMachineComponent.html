Building and destroying

StateMachineComponent.__call ()
Builds a new StateMachineComponent with no states.

Returns:

    StateMachineComponent The newly created StateMachineComponent.
StateMachineComponent:addTo (container)
Adds this component to a GameObject. This method registers the start, update and lateUpdate methods with the container GameObject. These methods will also be called on the current state, if they exist. The StateMachineComponent will be added as the stateMachine field of the GameObject.

Parameters:

  • container GameObject The GameObject this component is being added to.
StateMachineComponent:removeState (stateName)
Removes a state from the StateMachine, leaving it with no owner.

Parameters:

  • stateName string The name of the state to be removed. If there is no state with such name in the StateMachine, nothing is done.
StateMachineComponent:start ()
Starts the state machine, changing to the initial state. If the initial state has a start method, it will be called.
StateMachineComponent:lateUpdate (dt)
Executes the current state’s lateUpdate method, if present. Additionally, checks for conditions and executes any possible state change. If two or more transitions are possible, the one with higher priority is taken (or the one that was added first, if there is more than one transition with the same priority). State transition conditions take a single argument: the current state.

Parameters:

  • dt number The elapsed time since the last lateUpdate call, in seconds.
StateMachineComponent:changeToState (nextState)
Executes a transition to a specified state. The current state’s onExitTo and the target state’s onEnterFrom are executed, if found.

Parameters:

  • nextState State The target state.
StateMachineComponent:getStateFlags ()
Returns the current state’s flags.

Returns:

    table The flags, as an array of flags that can be true or false.
StateMachineComponent:update (dt)
Executes the current state’s update method, if present.

Parameters:

  • dt number The elapsed time since the last update, in seconds.
