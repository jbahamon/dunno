Class data.core.Component.TransformComponent

A Component that represents an element’s position and orientation.

Every GameObject has a TransformComponent.

Building and destroying

TransformComponent.__call ([position], facing) Builds a new TransformComponent.
TransformComponent:addTo (container) Adds this component to a GameObject.

Positioning and dynamics

TransformComponent:move (displacement) Moves the component by a certain amount, in pixels.
TransformComponent:moveTo (position) Moves the component to a certain position, in pixels.
TransformComponent:turn () Flips the component horizontally.


Building and destroying

TransformComponent.__call ([position], facing)
Builds a new TransformComponent.

Parameters:

  • position vector The starting position for the TransformComponent, as a vector. Optional (default is (0, 0))
  • facing number The starting facing for the TransformComponent. A facing value greater than zero means facing right; lesser than zero means facing left. Optional (default is facing right).

Returns:

    TransformComponent The newly created TransformComponent.
TransformComponent:addTo (container)
Adds this component to a GameObject. This method registers the move, moveTo, turn and changeToState methods with the container GameObject. The TransformComponent will be added as the transform field of the GameObject.

Parameters:

  • container The GameObject this component is being added to.

Positioning and dynamics

TransformComponent:move (displacement)
Moves the component by a certain amount, in pixels.

Parameters:

  • displacement vector The displacement to be applied, in pixels.
TransformComponent:moveTo (position)
Moves the component to a certain position, in pixels.

Parameters:

  • position vector The target position.
TransformComponent:turn ()
Flips the component horizontally.
