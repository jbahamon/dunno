Class data.core.Component.BaseComponent

The base class for all BaseComponents of a GameObject.

A Component represents a distinct functionality and/or characteristic of a GameObject. See other classes of the Component module for examples.

BaseComponent:addTo (container)
Adds this BaseComponent to a GameObject. When defining your own Basecomponents, you should register the Basecomponent’s methods with the container object in this method. Remember calling the BaseComponent’s (or the corresponding superclass) addTo method if you do this!

Parameters:

  • container The GameObject this Basecomponent will be added to.
