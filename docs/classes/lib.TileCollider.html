Class lib.TileCollider

A module that processes collisions between GameObjects and stage tiles.

Methods

Methods
lib.TileCollider:__call (stage)
Builds a new TileCollider with no elements, based on a given Stage.

Parameters:

  • stage The Stage object that represents the collidable world.

Returns:

    The newly created TileCollider.
TileCollider:addShape (element)
Adds an Element to the TileCollider. The TileCollider is NOT added in any way to the Element. Nothing is done if the Element was already added to the TileCollider.

Parameters:

  • element Shape The HardonCollider shape to be added to the TileCollider.
TileCollider:remove (element)
Removes a shape from the TileCollider. The TileCollider is NOT removed in any way from the Shape or other objects. Nothing is done if the Element was not previously added to the TileCollider.

Parameters:

  • element Shape The HardonCollider shape to be removed from the TileCollider.
TileCollider:update (dt)
Updates and checks tile collisions for all elements. data.core.Component.CollisionComponent’s onTileCollide (or the appropriate override) is called for each element touching a tile, for every tile it is colliding with.

Parameters:

  • dt number Time since the last update, in seconds.
