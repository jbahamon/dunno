Parameters:

    State The newly created State.
State:addTransition (condition, targetState[, position=1])
Adds a transition to the state.

Parameters:

  • condition function The condition function that shall be evaluated to determine if the transition should be performed.
  • targetState string The name of the state the transition will lead to.
  • position number The priority of the transition with respect to the others. (1 meaning the new transition will be checked before all those added before) (default 1)
